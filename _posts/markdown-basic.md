```

## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
###### Heading 6
```

#### Heading 4

### Bold

```
**This text is bold** 
```

__This text is also bold__`

### Italic

```
*This text is italic*
```

_This text is also italic_

### Bold and Italic

```
***This text is bold and italic***
```

___This text is bold and italic___

### Lists

#### Unordered List

```
- Item 1 
- Item 2 
	- Subitem 1 
	- Subitem 2
```

- Item 1 
- Item 2 
	- Subitem 1 
	- Subitem 2
#### Ordered List

```
1. First item
2. Second item
3. Third item
```

1. First item
2. Second item
3. Third item
