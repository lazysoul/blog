[Angry IP Scaner](https://angryip.org/)는 빠르고 가벼운 IP 주소 및 포트 스캐너입니다.

- 모든 범위의 IP 주소와 포트를 스캔할 수 있으며, 크로스 플랫폼을 지원합니다.
- 설치가 필요 없어 어디서나 자유롭게 복사하여 사용할 수 있습니다.

동시에 많은 Threads를 생성하여 IP 주소에 Ping을 보내 해당 호스트가 활성 상태인지 확인한 후, 선택적으로 호스트 이름과 MAC 주소를 조회하고 포트를 스캔하는 등의 작업을 수행합니다. 플러그인을 통해 각 호스트에 대한 수집 데이터를 확장할 수도 있습니다.
